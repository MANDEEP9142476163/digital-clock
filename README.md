# Modern Digital Clock

A responsive, dependency-free digital clock built with HTML, CSS, and JavaScript.

## Features

- Live hours, minutes, seconds, day, and date
- 12-/24-hour time toggle, remembered in `localStorage`
- Light/dark theme toggle, remembered in `localStorage`
- Time-based greeting and blinking time separators
- Fullscreen clock mode
- Responsive, glass-style interface

## Run it

Open `index.html` in any modern browser. No installation or build step is needed.
